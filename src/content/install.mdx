# Install

## Requirements
- Node.js 18+
- pnpm or yarn
- Hardhat or Foundry
- Git

## Repositories
- **Contracts:** https://github.com/sylanproject/sylan-contracts
- Future: **sylan-node**, **sylan-provider**

## Scripts
```bash
pnpm i
pnpm dev
```
For contract development, see [Contracts / Overview](/contracts/overview).
