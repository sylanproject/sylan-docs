# Welcome to Sylan

Sylan is a **decentralized API marketplace**. API Providers list services and set pricing (Subscription or PayPerCall). Consumers pay in **SYL** and receive responses from decentralized nodes. Fees are split among **Provider / Nodes / Platform** in basis points (BPS) summing to **10,000**.

## Value Proposition
- **Monetize data/APIs** without centralized intermediaries.
- **Trust minimization:** on-chain escrow & consensus.
- **Composable** with EVM tooling (Wagmi, viem, Hardhat/Foundry).
