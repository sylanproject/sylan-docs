# Core Concepts — Overview

Sylan coordinates **on-chain escrow & consensus** with **off-chain nodes** to deliver API responses.
- **PayPerCall (PPC):** one-off payments per request.
- **Subscription:** pre-paid time window or allotment.
- **Fees:** BPS split totaling 10,000 between Provider / Nodes / Platform.
